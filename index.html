常用命令

  1. ctrl c: 取消命令，并且换行
  2. ctrl u: 清空本行命令
  3. tab键: 可以补全命令和文件名，快速按两下tab键，可以显示备选选项
  4. ls: 列出当前目录下所有文件，蓝色的是文件夹，白色的是普通文件，绿色的是可执行文件
  5. ls -a: 显示全部文件包括隐藏文件
  6. ls -l: 显示文件的详细信息
  7. la：显示当前路径下面的所有文件 与 ls -a几乎一样
  8. pwd: 显示当前路径
  9. cd XXX: 进入XXX目录下, cd .. 返回上层目录
  10. cp XXX YYY: 将XXX文件复制成YYY，XXX和YYY可以是一个路径
  11. cp XXX YYY -r：若给出的源文件是一个目录文件，此时将复制该目录下所有的子目录和文件。
  12. mkdir XXX: 创建目录XXX
  13. mkdir XXX -p：确保目录名称存在，不存在的就建一个。
  14. rm XXX: 删除普通文件
  15. rm XXX -r: 删除文件夹
  16. rm XXX -f: 直接删除
  17. mv XXX YYY: 将XXX文件移动到YYY，和cp命令一样，XXX和YYY可以是一个路径；重命名也是用这个命令
  18. touch XXX: 创建一个文件
  19. cat XXX: 展示文件XXX中的内容
  20. adduser 用户名: 创建用户
  21. usermod -aG sudo 用户名: 给用户分配sudo权限
  22. userdel -r 用户名: 删除一个用户
  23. su : 切换用户
  24. ps -a: 显示所有用户的进程
  25. top: 显示所有进程
  26. kill -9 pid: 杀死一个进程
  27. apt-get install 软件名: 安装软件
  28. clear 快捷键 Ctrl+l: 翻页
  29. tar -zcvf XXX.tar.gz 目录： 将目录直接打包压缩为”.tar.gz”格式
  30. tar -zxvf tmp.tar.gz: 将”tmp.tar.gz”解压到当前文件夹
  31. netstat -tunlp: 用于显示 tcp，udp 的端口和进程等相关情况。
  32. grep 'XXX' 文件：从文件中查找＂XXX＂字符串　
  33. netstat -tunlp | grep 端口号: 查看端口占用语法格式：
